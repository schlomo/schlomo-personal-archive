  • DHCP & TFTP Server on a single 1.44" diskette. This MS-DOS formatted diskette contains a complete DHCP and TFTP server and has ca. 400 KB free for files to be served via TFTP. All the configuration (network setup and DHCP configuration) is done via text files on the diskette and can be done from any PC. For remote control of the server a Web-management interface is included. Documentation.
  • POP3 PAM Module Authenticates users against a POP3 server or similar. Documentation.
  • LABMon Monitors activity in computer labs by pinging all computers periodically and creating HTML tables and PNG pictures with usage statistics. It also collects statistics for later interpretation and aggregation (aggregation module is not yet written). It can also create a map of available computers in a computer lab, e.g. to be placed in the entrance and show where are available computers. Perl script which can also run on Perl/Cygwin.

    The aim is to create usage statistics for the system administrators and a web-based tool for users to find and locate available computers Documentation Sample Config file Live Example.
