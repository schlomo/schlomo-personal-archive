NDS Object Creation Etiquette

1. Naming

All NDS objects must be named in Latin letters (preferrably in English).

All NDS objects should have clear, describing names without spaces or special characters in them (The Common Name of the object can be different from the Display Name in NAL):

Good Examples: Load_NALDesk
Remove_NAL
Install_Office_97
Install_NAV_Update_0899
Bad Examples: test
kuku
temp
load
math program

Similar Objects should have similar names (with the similar part at the beginning), if they contain numbers then the numbers should be at the end of the name.

Good Examples: Student_1
Student_2
Student_3

Rishum_Student
Rishum_Oved
Rishum_Internet_Shnaton
Rishum_Internet_Student_Info
Bad Examples: First Student
Second Student
Third Student

Student Rishum
Oved Rishum
Shnaton
Internet Student Info
