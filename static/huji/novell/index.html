Novell Netware Programs

UpdateWS is a system to automatically update computers from the network. Includes a computer image version control system. Documentation.
Extract MAC Address. This tools helps to extract the MAC address from the output of an ODI NIC driver for batch processing. Documentation.
Print Job Manager is a PCONSOLE replacement to let users & admins manipulate print jobs. Screen-shot.
Change Novell Password. See Screen-shot.
Account Information displays a host of useful information to the user & has 6 customizable buttons.Screen-shot. Documentation.
Initialize User displays text to users when it changes. Also allows to run a program once per user. See Documentation for further explanations.
Check Password let's you check other users' passwords without logging in. Displays basic information and can unlock accounts locked by Intruder lockout. This is NOT a hacker tool, since it triggers the Intruder Lockout. See Screen-shot.
